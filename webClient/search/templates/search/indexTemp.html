Search Results

This panel shows the search results. When you click one of the results, a new tab will open showing you the document you selected. The icons on each result represent the origin search engine:

  • Bing:
  • Google:
  • Faroo:
  • Carrot2 (eTools Metasearcher)

To make a new query, you can use the search bar on top of the page. In adition to this, ZaexSearch offers new posibilities for improving your search. These posibilities can be accessed via the buttons on the left of the bar below the search form.

  1. Relevance Feedback: This feature adds a word to your query, based in the previous queries you have made.
  2. Diversity: This feature reorders the results, to prevent similar results from appearing near each other in the rank

Additionally, you can filter your results by using the checkboxes located on the right of the page, selecting the search engines.

Clicked results

This panels shows the search results that you have selected in this and other related queries.

Search session

You can save documents to read them later. For that, in the Search Results tab, you can drag the result you want to save to the save icon located in the bottom of the screen. Saved results will appear in green.

Additionally, they will appear in the Search Session tab. In this tab, you can do the following actions:

  1. Delete results: To delete results, drag the result you want to delete to the trash icon located in the bottom of the screen
  2. Save result list: You can save the results, and start a new list. To do that, you have to introduce a name for the session in the corresponding form. In case a problem has happened, it will be notified under the save form
  3. Open result list: You can restore a previous list in case you have saved it on a computer with the same IP and the same browser. To do that, introduce the name in the corresponding form
